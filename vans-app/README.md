# Van's App — Planning Session

Interactive planning form for building a mobile app for independent hairstylist Van (San Jose, CA).

**The app goal:** Minimal typing, works behind the chair with gloves, aesthetically pleasing — a better alternative to Gloss, Vish, and SalonScale.

## Modules under consideration

1. **Formulas** — Track hair color formulas with ml/gr/oz units
2. **Inventory** — Barcode scanning, stock tracking, low-stock SMS
3. **Client Info** — Name, number, photos, notes
4. **Profitability** — Cost per service, break-even analysis

## The Premortem

We ran a premortem analysis on this project. Key findings:

- **Most likely failure:** The app is built for one stylist but marketed to all
- **Most dangerous failure:** Profitability numbers are wrong because fixed costs aren't tracked
- **Hidden assumption:** One person's workflow description is a valid proxy for how all stylists work (it's not)
